Projects

In my free time, I like to create things. Below are some of them! Most of these projects are open-source on Github.

  • Sudoku, a sudoku puzzle generator and solver. Motivated both by the UI challenge and to learn how sudoku puzzles are generated.
  • Maze factory, one of my earlier experiments with the HTML5 canvas. It draws a DFS-generated maze to the screen live in front of the user.
  • Budget Tracker, built to allow roommates and friends to easily track and manage collaborative budgets. Built on a Python + Flask backend.
  • General Cluster, an application for displaying data and finding clusters using k-means clustering. Built on Java.
